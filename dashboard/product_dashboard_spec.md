# Product Dashboard Specification

## Executive Questions
1. Is acquisition growing?
2. Which channels convert best?
3. Where is the biggest funnel drop-off?
4. Are new user cohorts retaining better over time?
5. Did the experiment improve purchase conversion enough to ship?

## Dashboard Tabs

### 1. Executive Overview
- total visitors
- total purchasers
- purchase conversion rate
- total revenue
- average DAU
- experiment lift
- day-7 retention

### 2. Funnel Analysis
- funnel volumes by stage
- conversion rate by channel
- conversion rate by device
- drop-off heatmap by segment

### 3. Retention
- cohort table
- day-1 / 7 / 14 / 30 retention trend
- retention by acquisition channel

### 4. Experimentation
- control vs treatment conversion
- lift and confidence interval
- significance flag
- recommendation banner

## Recommended Dashboard Tools
- Tableau
- Power BI
- Streamlit (for GitHub demo)

## Recommended Storyline for Recruiters
- Start from business objective
- Show KPI roll-up
- Walk through biggest funnel loss point
- Show which segment underperforms
- End with experiment recommendation
